This project contains three KiCAD files: one for the autopilot panel, one for the radio panel and one for the overhead panel. The controller is an Arduino Mega 2560 for the autopilot and radio panels. A Mega 2560 Pro Mini is used for the overhead panel. Framework: MobiFlight. Some of the electronic components that have been used are listed in the BOM_FlightSim.xlsx.
